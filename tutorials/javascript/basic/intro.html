Getting started with JavaScript

In this article you will learn how to run JavaScript on your computer

Before that a quick intro to JavaScript:

Javascript is a powerful programming language that makes website interactive. JavaScript was invented by Brendan Eich.

JavaScript was initially created to “make web pages alive”. The programs in this language are called scripts. They can be written right in a web page’s HTML and run automatically as the page loads.

JavaScript can execute not only in the browser, but also on the server, or actually on any device that has a special program called the JavaScript engine.

How to use JavaScript

Internal JS: JavaScript can be embedded into HTML file using script tag, and then script tag can be used inside head tag or body tag.
External JS: JavaScript can be written in another file and then link the file in the head tag of the HTML file

How to run JavaScript code in Browser

To run JavaScript script in browser open any browser and right click and select Inspect(or F12 for chrome).
Go to console and write JavaScript code and press enter to execute the script.


How JavaScript Engine works

The engine(embedded in browser) reads the script.
Then it converts script to machine language.
And then machine code runs fasts.

Hello world

console.log("Hello World!")

'Hello World!'
