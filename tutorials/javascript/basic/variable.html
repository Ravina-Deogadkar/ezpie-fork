JavaScript Variables

In this article you will learn what's a variable

What's a variable?

A variable is a label that reference a variable like a number or string. We need to declare a variable before using them.

How to declare a variable

To declare a variable we need to use let keyword along with variable name.
A variable name can be any valid identifier. We can also use var keyword to declare a variable both are similar but var uses "old fashion" way. We would look at them later in our tutorial.

# This is a variable called message with a value "Hello There!"
let message = "Hello There!"

Here we have declared a variable called message with a value of "Hello There!".
Take it in the simple way message is just a label referencing to the location where "Hello There!" is stored.
So in that manner, label can reference to another location right?
We will make message variable reference to some other String.

let message = "Hello There!"
message = "Have a great day!"

message is now reference to other string "Have a great day!" but what if instead of string other data type is assigned to message. Will it throw error?
No!It's beacause JavaScript are dynamically typed that means data type of a variable depends on data it reference. Although it is not recommended, but varaibles in JavaScript can reference any type of data .

NOTE: Remember one thing, the variable name are case sensitive

Undefined

Undefined is a special data type in JavaScript it's not like null or undeclared. When we declare a variable and do not assign any value to it, the value of variable eventually becomes Undefined. example:

let counter;
console.log(counter)

Here counter is declared but value is not assigned to it and Therefore value of counter is Undefined. Undeclared on the other hand is when the variable is used even before we declare it. We get undeclared exception in the log when we attempt to access variable before declaring them. Null is an valid data type that can be assigned to a variable.

How to name a variable

do and don't to name a variable in python:

  • Variable name must not start with a number(1name, 23name)
  • Variable name must start with a letter, underscore or dollar sign(name, _name, $name)
  • Variable name can only contain number, letter, underscore or dollar sign(name1, n1ame)
  • Variable name must not have space in between(my age)
  • Variable name cannot use reserved words(const, class, var)

# Legal ways
let name = 'Tom'
let name_first = 'Tom'
let $name1 = 'Tom'
let _name1 = 'Tom'

# Illegal ways
let 1name = 'Tom'
let name-first = 'Tom'
let name first = 'Tom'
let const = 'Tom'

multi-word naming
